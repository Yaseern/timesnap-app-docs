TimeSnap is a Jira Forge app that extends Jira worklogs with dashboards and Excel export.

## What data we access
- Jira worklogs (time spent, issue, project)
- User information for dashboards (display name, account ID)
- Minimal app configuration in Forge storage (KVS)

## What data we store
- App settings (e.g., custom fields)
- Feature flags
- No user credentials or external data

## Data usage
- Data is only used **inside Jira** to show dashboards, worklogs, and exports.
- **No data is sent outside Atlassian**.
